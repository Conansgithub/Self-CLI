# superagents Examples

本目录（本文件同级）提供可复制的端到端示例（命令序列 + 期望产物 + 常见失败点）。

- `examples-01-spec-first.md`：先写规范（Spec-first）再进入变更闭环
- `examples-02-change-plan-run.md`：标准变更闭环（change → plan → run → check → archive）
- `examples-03-high-risk.md`：高风险变更（risk_level=high/critical）的门禁与证据链
- `examples-04-ci-integration.md`：CI/本地统一检查入口（sa_check）
- `examples-05-migration.md`：旧文档升级/补字段（sa_migrate）与环境自检（sa_doctor）
